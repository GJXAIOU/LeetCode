## 题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

```java
public class Solution {
   static int res = 0;
    public int JumpFloor(int target) {
        if(target == 1 || target == 2){
            return target;
        }else{
            return res + JumpFloor(target - 1) + JumpFloor(target - 2);
        }
    }
}
```
